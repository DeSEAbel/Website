You can get involved in DeSEAbel in 3 different ways

Improve the source code of the DeSEAbel app.

You can find the code on our GitHub repository: https://github.com/DeSEAbel/deseabel

All contributions are welcome, from bug fixes to new features. You can for example:

  • Add a new language to the app to improve the user interface
  • Add new marine fauna data
  • Adapt the sound propagation model
  • And many more!
